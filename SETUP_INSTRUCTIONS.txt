CASH NOW PHILLY - SETUP INSTRUCTIONS
=====================================

Prerequisites:
- Node.js 18+ installed
- Google Cloud account with Sheets API enabled
- Google Sheets spreadsheet created

STEP 1: Install Dependencies
-----------------------------
npm install

STEP 2: Google Sheets Setup
----------------------------
1. Create a Google Cloud Project:
   - Go to https://console.cloud.google.com/
   - Create a new project or select existing one
   - Enable Google Sheets API

2. Create Service Account:
   - Go to IAM & Admin > Service Accounts
   - Create a new service account
   - Grant it "Editor" role
   - Create a JSON key and download it

3. Create Google Spreadsheet:
   - Create a new Google Sheet
   - Add headers in first row: Timestamp, Address, Phone Number, Source, User Agent, IP
   - Share the sheet with the service account email (found in JSON key)
   - Copy the Spreadsheet ID from the URL (between /d/ and /edit)

STEP 3: Environment Variables
------------------------------
1. Copy .env.example to .env:
   cp .env.example .env

2. Fill in the following values:

GOOGLE_SHEETS_SPREADSHEET_ID=
   Your spreadsheet ID from Google Sheets URL

GOOGLE_SERVICE_ACCOUNT_EMAIL=
   From the downloaded JSON key file (client_email field)

GOOGLE_PRIVATE_KEY=
   From the downloaded JSON key file (private_key field)
   Keep the quotes and newlines as-is

NEXT_PUBLIC_SITE_URL=
   Your production URL (e.g., https://cashnowphilly.com)

NEXT_PUBLIC_PHONE_NUMBER=8669719529
NEXT_PUBLIC_PHONE_DISPLAY=(866) 971-9529

NEXT_PUBLIC_GTM_ID=GTM-TRJQWKSB
NEXT_PUBLIC_GOOGLE_ADS_ID=AW-728002925

STEP 5: Run Development Server
-------------------------------
npm run dev

Visit http://localhost:3000

STEP 6: Build for Production
-----------------------------
npm run build
npm start

STEP 7: Deploy
--------------
Recommended platforms:
- Vercel (easiest): vercel --prod
- Netlify: Connect GitHub repo
- AWS/DigitalOcean: Use Docker or PM2

Environment Variables on Vercel:
1. Go to Project Settings > Environment Variables
2. Add all variables from .env
3. Redeploy

TESTING:
--------
1. Fill out the form with test data
2. Check your Google Sheet for the new row
3. Verify tracking pixels fire (use browser dev tools)
4. Test on mobile devices

MAINTENANCE:
------------
- Monitor Google Sheet for leads
- Check error logs regularly
- Update dependencies monthly: npm update
- Keep SSL certificates current

TROUBLESHOOTING:
----------------
Form not submitting:
- Check Google Sheets API is enabled
- Verify service account has access to sheet
- Check private key format (newlines must be \n)

Images not loading:
- Verify all images are in /public/assets/
- Check image paths in components
- Run: npm run build to see any errors

SEO issues:
- Use Google Search Console to verify indexing
- Check robots.txt is not blocking
- Verify meta tags with Facebook Debugger
