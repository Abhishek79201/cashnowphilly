API ROUTES:
===========

POST /api/submit-lead
- Accepts: address, phoneNumber
- Validates input
- Saves to Google Sheets
- Returns success/error
- Includes timestamp, source, user agent, IP

STYLING APPROACH:
=================

1. Tailwind utility classes for layout
2. Custom colors in config:
   - primary: #0F526B (blue)
   - accent: #C51317 (red)
   - gray-light: #F3F4F8
3. Custom components:
   - btn-primary (red button)
   - section-container (max-width wrapper)
   - input-field (form inputs)
4. Font families:
   - Lato: headings
   - Roboto: body text
5. Responsive breakpoints:
   - mobile: 600px and below
   - tablet: 601-1024px
   - desktop: 1024px and above

- NEXT_PUBLIC_SITE_URL

Optional:
- NEXT_PUBLIC_GTM_ID

DEVELOPMENT WORKFLOW:
=====================

1. Clone repository
2. npm install
3. Copy .env.example to .env
4. Fill in environment variables
5. Add assets to /public/assets/
6. npm run dev
7. Test form submission
8. npm run build
9. npm start (test production build)
10. Deploy to Vercel/Netlify
